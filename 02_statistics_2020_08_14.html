Test for comparison between means of CONABIO and GMW

Assumptions for paired t test are as follows:

  1. The sampling units are randomly sampled from the population

  2. The paired differences have a normal distribution in the population

hist(data$Area_CONABIO - data$Area_GMW_calculated, xlim=c(-30,10),
     main = "")

Not normally distributed so we used the wilcoxon test. (Figure 1B)

wilcox.test(data$Area_CONABIO, data$Area_GMW_calculated, paired = TRUE) #used because we have a non normal distribution

Root Mean Squared Estimate Calculation (RMSE)

The RMSE is used to understand how much the satellite measurements of the same area of mangrove differ from the drone measurements of the same area.

d2 <- data_drone %>% 
  select(UID, TotalFishnetArea_1, Area_ha_drone:Area_ha_GMW) %>% 
  rename(CONABIO = Area_ha_CONABIO, 
         GMW = Area_ha_GMW) %>% 
  gather(key = "key", value = "Area", CONABIO:GMW) %>% 
  mutate(difference = abs(Area_ha_drone - Area))
# Using the full dataset 
d4 <- d2 %>% # General mean squared difference 
  mutate(squared_residual = difference^2) %>% 
  group_by(key) %>% 
  summarise(RMSE = sqrt(mean(squared_residual)))
## `summarise()` ungrouping output (override with `.groups` argument)
# Square of the standard error 
kable(d4, caption = "The Root Mean Square Estimates") %>%
